# photo — 個人與社群照片

| 檔名 | 用途 | 建議尺寸 |
|---|---|---|
| `portrait.jpg` | 首頁個人照(直式) | 約 600×750px(4:5) |
| `npo_photo.jpg` | Community 頁 NTUMEIC 標誌或主照(方形) | 約 800×800px(1:1) |
| `community_1.jpg` ~ `community_6.jpg` | Community 頁「Activities」活動照片牆(橫式) | 約 800×600px(4:3) |

活動照不必放滿 6 張;沒放的會顯示佔位框,想減少格子數請告訴 Claude 或直接刪 `index.html` 裡對應的 gallery 區塊。
